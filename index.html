Numerophile

Choose a folder:

Number all files

This will rename all files in the chosen folder with a (123)_ prefix, starting at your selected number.

Any files which already have a (123)_ prefix will be left alone*




*Unless padding with a zero is required to keep files in the same "alpha-numerical" order.

Advanced Options

These options have the capability to completely destroy your numbering system. Do not use them if you do not know exactly what you are doing.

De-number all files

This will remove all "(123)_" prefixes from all files in the chosen range (inclusive)



Numbering override

Example: By default, if there is a file with a prefix of (100)_, and you choose a starting value of 99, the program will not allow you to continue, as this will cause you to have two files with the same "unique" ID

This option overrides that behaviour.
